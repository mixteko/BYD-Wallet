# BYD Wallet

PWA mobile-first para BYD King con tema oscuro premium y enfoque Tesla/BYD.

## MVP incluido

- Dashboard principal
- Registro de gasolina
- Registro de cargas eléctricas
- Mantenimiento
- Historial consolidado
- Reportes mensuales
- Estadísticas con gráficas interactivas

## Métricas incluidas

- Gasto diario
- Gasto semanal
- Gasto mensual
- Gasto anual
- Costo por km
- km/L
- km/kWh
- Rendimiento promedio
- Ahorro acumulado

## Arquitectura preparada para

- OCR de tickets
- Integración OBD2
- Múltiples vehículos

## Stack

- React 19
- TypeScript
- Vite
- vite-plugin-pwa
- Recharts
- Lucide React

## Desarrollo

```bash
npm install
npm run dev
```

## Validación

```bash
npm run lint
npm run build
```
